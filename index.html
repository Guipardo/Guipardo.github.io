Sobre mim


Estudo Sistemas de Informação , gosto de desenvolver sites trabalhando na parte de Front End , onde posso expressar minha criatividade desenvolvendo interfaces simples, bonitas e fáceis de utilizar, sempre focando na experiência dos usuários - UI/UX.


Nós programadores gastamos grande parte do tempo lendo código fonte do que escrevendo, assim sempre busco fazer um código limpo e de fácil entendimento , pois é fundamental durante o ciclo de desenvolvimento.


Trabalho também para que as páginas de meus sites possam ser bem vizualizadas em qualquer dispositivo, focando na responsividade.

Objetivos


Interação

Trabalhar no desenvolvimento de conteúdo, layouts e interação com o site utilizando HTML, CSS e Javascript

Responsividade

Trabalhar modificando layouts para que sejam vizualizados corretamente em dispositivos móveis



Performance e Search Engine

Procurar soluções para desenvolver sites que sejam rápidos e fáceis de se encontrar pelos mecanismos de busca
